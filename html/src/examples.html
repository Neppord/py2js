Examples¶

All examples are in the examples directory.

Game of Life¶

online: gol.html source: examples/gol.py

Uses the HTML5 Canvas to draw the game of life.

ExtJS Demo¶

online: ui.html source: examples/ui.py

Shows how to call the ExtJS library from Python to create a nice GUI in the browser.

Triangulation Demo¶

online: triangulation.html source: examples/triangulation.py

Shows a real life example of a Python script that triangulates a domain, together with a simple canvas plotting. You can see that the Python output and JavaScript output is identical.
